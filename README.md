# via-app-zh

## 关于 VIA

VIA 是一个强大的、开源的基于网络的界面，用于配置你的 [QMK](https://qmk.fm) 机械键盘。它允许你自定义你的键位映射、创建宏，以及动态调整 RGB 设置（如果它有 RGB），而无需重新编译你键盘的固件。这使得键盘定制更加容易和普及。

## 关于 VIA 中文版

本项目是 [the-via/app](https://github.com/the-via/app) 的非官方汉化版本，且不随原项目同步。

本项目的键盘定义来自于 [VIA 的键盘仓库](https://github.com/the-via/keyboards)

## 遇到了问题？

由于 [VIA 中文版](via-zh.top) 的汉化方式较为简单粗暴，可能会产生 [VIA 原版](https://www.usevia.app/) 没有的新 BUG，如果你遇到了影响正常使用的BUG，请先试试 [VIA 原版](https://www.usevia.app/)。

由于 [VIA 中文版](via-zh.top) 是非官方汉化版本，所以 VIA 不对本版本的 BUG 负责，请不要将本版本的 BUG 反馈至 VIA 原版，这会对 VIA 原版的开发者产生困扰。

## 开发&构建

`npm install`

安装项目依赖

`npm run dev`

在开发模式下运行应用。如果你进行编辑，页面将重新加载。 你还会在控制台中看到任何代码风格错误。

`npm run build`

将你的站点静态复制到 build/ 文件夹。 你的应用已准备好部署！
